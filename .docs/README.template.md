## Getting Started

To get started, clone the
[sbv2-evm](https://github.com/switchboard-xyz/sbv2-evm) repository.

```bash
git clone https://github.com/switchboard-xyz/sbv2-evm
```

Then install the dependencies

```bash
cd sbv2-evm
pnpm install
pnpm build
```
